Lesson 3 - Introducing Errors

Let's get started....

In the console, type:

cd ~/work
mkdir Errors
cd Errors

Now we have our working directory. Now , lets create our code file:

vim main.go

Type i to begin editing the file. Now, type the following:

package main

import (
"fmt"
)

func addition(x int, y int)int {
return x + y
}

func main() {
var s string = "three"
fmt.Println(addition(1,2))
}

Now save the file by hitting ESC key typing: :wq!

Now you can run your program by typing:

go run main.go

You should see an error complaining about string and int.

To fix this, change the following:

var s string = "three"

to

var s int = 3

Save and rerun your code.

Congratulations! You have started troubleshooting!
