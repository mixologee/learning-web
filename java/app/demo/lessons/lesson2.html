Lesson 2 - Introducing Types

Let's get started....

In the console, type:

cd ~/work
mkdir IntroTypes
cd IntroTypes

Now we have our working directory. Now , lets create our code file:

vim main.go

Type i to begin editing the file. Now, type the following:

package main

import (
"fmt"
)

func sayHello(s string) string {
return "Hello " + s
}

func main() {
fmt.Println(sayHello("Your Name"))
}

Now save the file by hitting ESC key typing: :wq!

Now you can run your program by typing:

go run main.go

You should see Hello Your Name returned.

You can also compile the code and run by command as follows:

go build main.go
./main

Congratulations! You have started working with strings and function calls!
