Welcome to Learning Go @ BlackRock!

Let's get started....

In the console, type:

cd ~/work
mkdir HelloWorld
cd HelloWorld

Now we have our working directory. Now , lets create our code file:

vim main.go

Type i to begin editing the file. Now, type the following:

package main
import ("fmt")
func main() { fmt.Println("Hello World!") }

Now save the file by hitting ESC key typing: :wq!

Now you can run your program by typing:

go run main.go

You should see Hellow World! returned.

You can also compile the code and run by command as follows:

go build main.go
./main

Congratulations! You have started learning Go!
